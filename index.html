My Pharmacist

This project took place at Open Data Hackathon that was held in 2015. That was also the first hackathon I have attended. I had really basic programming knowledge at that time, but hackathons looked so interesting to me that I needed to take place at such event.

Application "My Pharmacist" that we were developing, was about helping people find their medicines quick and easy. We made a big map where you can mark your location, and after that map pops up three nearest pharmacies in relation to that location. Second thing you could do is to type in specific medicine you are searching for, and app will list you all substitutes for that medicine, along with their prices and information. So people could find the same medicine for a smaller price. It was a simple application with not much functions, but unique and useful.

I was responsible for Javascript mostly. Team needed Google Maps, so I took that role.

  • Date: December 2015
  • Used: Javascript

Luxury Box

My friend needed to make a small e-commerce web site, but he also needed some PHP code for that. So I jumped in to take that part.

This is a web site where you can buy roses. User specify color (white, red, yellow, pink), quantity, and then just check out. After filling a form with his data like name, email, address for delivery, he just sends an order. User can also choose the way he wants to pay. We implemented 2checkout service, where he can pay with his credit card. After order has been sent, site owner gets an email with all informations needed to deliver that products.

  • Link: Luxury Box
  • Date: March 2016
  • Used: PHP

Education For Everyone

This project took place at Start Hackathon organized by FONIS.

The idea was to make a web site where kids with limited abilities can find a corresponding school. We needed to take care of colors and contrast on the web site, and also to simplify use of it as much as we can. Ministry of Education gave us a database filled with all schools in the country, as well as some additional information about them. What we didn't have is the data about what each school have to offer for kids with limited abilities. Well, there was some informations actually, but they were hidden in long school description texts. So we made a script to search for specific keywords that can tell us if school have some of the tools that can help those kids. After we did that, it was easy to fetch list of schools by some filter. User interface had the most important role in this project. We won 3rd place.

  • Date: May 2016
  • Used: PHP, MySQL

DelphiGuru

My colleague at Faculty of Organizational Sciences and me took this project in order to take opportunity to get more experience in our fields.

We had a task to make application for delphi analise. Group of experts can use this app to give their forecasts, and not just that, app will analise all input data and give corresponding diagrams and numbers.

Next part of the text I'm taking from documentation:

Introductory remarks:

The application is created using a technology whose primary purpose is creating software solutions to operate in a distributed environment such as the Internet. Consequently, application enables simultaneous access by multiple users.

Types of users on the system:

When user launches application, depending on the predefined privilege and access level, the application can be accessed as a normal user, a participant in a survey, or as a system administrator.

User manual:

By choosing one of the defined research from drop-down list, the user automatically becomes a participant in the study. In this case, it is only possible to input data into pre-defined forms. After entering all the data, user needs to save changes by selecting "Save", or click on the Undo button. The data is recorded, and user receives feedback that everything went fine. It is important to note that in this mode the user can not access the questionnaire or to make changes to previously saved answers. If the user chooses the option "System Administration", he gets all the rights managing the system. This option is intended for managers and participants of the Coordination Group. By choosing these options to be shown in a new window it provides various options such as view statistics and questionnaire replies, changes on the questionnaire and the creation of new questionnaires.

  • Link: DelphiGuru
  • Date: July 2016
  • Used: PHP, MySQL, Javascript

Ontime Limo and Dispatch

My friend and I had a job to develop online booking system for a passengers transportation company. There is a price for every route, from airport and to airport. User needs to provide informations about pick up and drop off address and time, as well as some personal and contact details. User can also choose some extras like child seats. After information has been provided, application calculates final price based on provided informations and chosen vehicle.

User needs to pay by credit card immediately after reservation and price calculation, after what he or she gets an email with reservation details.

  • Link: Ontime Limo and Dispatch
  • Date: June 2017
  • Used: Wordpress, PHP
